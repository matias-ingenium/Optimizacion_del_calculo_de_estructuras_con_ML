Explicación de los archivos

main -> contiene el hilo principal de ejecución (carga de datos o creación si no estan, crear, entrenar y testear la red)

mallado-> importa el mallado desde la url

POD-> interfaz encargada de la concexión con UQpy, tiene las 2 funciones para hacer POD

redes-> clase que contiene las características y funcionamiento de las operaciones relacionadas con redes

testing-> encargado de generar el conjunto de entrenamiento

training-> genera el conjunto de entrenamiento y matriz de base reducida

training2-> hace lo mismo que training pero obtiene los snapshots en paralelo

training_data.pkl contiene los archivos de datos necesarios para el entrenamiento de la red
    0-> 4000 datos 
    1-> 100 datos
    2-> 4000 datos con 28 elementos en la base reducida
    3-> 100 datos 
    4-> 4000 datos con 16 elementos en la base reducida
    5-> 100 datos con 16 en base reducida pero esta restringido la primer coordenada de mu a 1 y los demas valores estan en (1,20)
    6-> Lo mismo de 5 pero con 4000
    7-> 4000 pero se saca el 1 fijo de los parametros

testing_data.pkl contiene los archivos necesarios para testear la red
